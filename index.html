Welcome!

I am a data scientist and Ph.D. graduate in Astronomy from New Mexico State University. I currently work at Red Violet, which provides information services linking people, criminal, asset, and business data. This requires highly efficient and technical solutions to clean, process, and match together tens of terabytes of data.

For my doctoral research I worked with Professor Anatoly Klypin. My dissertation was to explore the techniques used to estimate the mass of galaxy clusters using weak gravitational lensing, using simulations from the MultiDark suite.

I'm always working on machine learning projects, if you're interested in those check out my Project Portfolio. I'm working on publishing everything I've worked on, so stay tuned as I add more.

If you are interested in weak lensing or simulation surface density codes, check out my codes page or my Github page.
